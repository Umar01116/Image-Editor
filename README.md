# Modern Image Editor

A fully functional **Image Editor** built from scratch using **HTML, CSS, and JavaScript**.  
This project allows users to **upload images**, apply **real-time filters**, use **preset effects**, and **download the edited image**. It features a **modern dark-themed UI/UX** with clean sliders, buttons, and preset controls.

- **Preset Filters:** Quickly apply predefined filters like B&W, Vintage, Cinematic, Soft, Dramatic.
- **Reset Function:** Reset all filters to default.
- **Download Image:** Save your edited image as PNG.
- **Modern UI/UX:** Clean dark theme, responsive layout, hover effects, rounded buttons, polished sliders.
